## ThystTV 1.1.6

### Highlights
- Hotfix release for a regression where switching to a different live stream could immediately close the player on the first attempt.
- Fixed Media3 sleep timer handoff behavior so opening a new stream no longer triggers an unintended instant shutdown.
- Improved player replacement cleanup so the active player is closed before a new one takes over.

### Included In This Release
- Added a read-safe sleep timer lookup path instead of using the timer setter as a query.
- Explicitly clear the service sleep timer when the player is closed.
- Close the existing player fragment before replacing it during stream handoff.

### What To Watch
- Switching between live streams from Popular, category pages, and channel pages.
- Closing a stream and immediately opening another stream.
- Sleep timer behavior during player close and stream-to-stream transitions.

### Release Type
- Hotfix release focused on playback stability.
