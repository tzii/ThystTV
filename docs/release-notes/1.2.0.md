# ThystTV 1.2.0

## Highlights
- Player polish focused release for a smoother everyday viewing experience.
- Cleaner repo front door with maintainer docs, release notes, and safer release workflow.
- Better foundation for ThystTV-owned releases instead of relying on upstream Xtra defaults.

## Player improvements
- The fullscreen playback speed control now shows the active speed.
- Fullscreen VoD scrubbing is duration-aware so long videos can be traversed much faster.
- Minimized player presentation was cleaned up to avoid broken-looking black bars on wide aspect ratios.

## Visual refresh
- README and site screenshot slots are prepared for final 1.2 visuals.
- GitHub/repo presentation docs are ready for final screenshots and branding assets.

## Fixes
- Restored the stock quality popup after the custom redesign caused portrait-mode regressions.
- Added debug signing support for local and CI debug APK generation.

## Known issues
- Final screenshots still need to be captured and added before release.
- The update checker still needs a ThystTV-specific pass for version comparison, changelog display, and release download progress.
- Latest Xtra commits should be synced selectively rather than merged wholesale.
